开发者视角：命令一行不变

开发者实际敲的命令（3 条）

/opsx:propose "需求"

5 秒生成 proposal/specs/tasks（自动 5 步任务）

/opsx:apply

Claude Code 自动跑完 5 步闭环（5–15 分钟）

/opsx:archive

PR merge 后归档 spec 成 SSOT

体感：和裸 OpenSpec 完全一样

不用学新命令、不用记新步骤

apply 阶段内部展开（自动）

Task 1 implement

Generator Agent 写 src/

Task 2 test-write + Task 3 test-run

Generator 写 happy path + 跑通

Task 4 evaluator ★

test-evaluator subagent 独立 context 补测试

Task 5 mutation ★

Stryker 跑 → 不达标循环回 Task 4
